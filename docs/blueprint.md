# **App Name**: eComVoyage

## Core Features:

- Product Browsing: Allow users to browse products by category and view detailed product information.
- Product Search: Enable users to search for products using keywords.
- Google OAuth: Implement user authentication using Google OAuth for easy login.
- Shopping Cart Management: Allow users to add products to a shopping cart, update quantities, and remove items.
- Secure Checkout: Implement a secure checkout process for users to purchase items in their cart.
- Recommended Products: AI-powered tool which recommends products based on browsing history and items in the shopping cart, offering personalized suggestions to enhance the shopping experience.

## Style Guidelines:

- Primary color: Light blue (#A7D1F7) to evoke trust and serenity, complementing the e-commerce experience.
- Background color: Very light blue (#F0F8FF) for a clean, unobtrusive background.
- Accent color: Medium blue (#7CA3CC) for interactive elements and call-to-action buttons, creating clear visual cues.
- Body and headline font: 'Inter' for a modern, clean and readable aesthetic throughout the website.
- Use simple, consistent icons for navigation and product features.
- Employ a clean and professional layout with clear sections for products, cart, and checkout.
- Incorporate subtle animations for product transitions and user interactions.
